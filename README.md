# 房屋房价线性回归预测项目
## 一、项目概述
本项目基于Python实现经典线性回归机器学习建模，模拟房产多维度特征数据集，完成完整机器学习标准化流程：数据加载、数据探查、相关性分析、数据集划分、模型训练、多维度模型评估、可视化输出。
适用于大数据/数据科学专业学习作品集，体现基础机器学习建模、数据预处理、特征分析能力。

## 二、技术栈
1. 数据处理：Pandas、NumPy
2. 可视化绘图：Matplotlib、Seaborn
3. 机器学习建模：Scikit-learn LinearRegression
4. 运行环境：Windows Python虚拟环境

## 三、数据集说明
数据集包含4个特征、1个预测目标：
- area：房屋建筑面积（㎡）
- room_num：卧室数量
- floor：房屋楼层
- subway_dist：距离地铁站距离（km）
- price：房屋总价（万元，预测目标标签）

## 四、完整功能流程
1. 自动生成结构化房产数据集，导出本地CSV文件；
2. 数据探查：打印样本前5行、统计指标、缺失值检测；
3. 数据预处理：区分特征与标签，划分80%训练集、20%测试集；
4. 线性回归模型训练，输出各特征权重、模型截距；
5. 多维度模型效果评估：MSE均方误差、MAE平均绝对误差、R²拟合优度；
6. 生成4张专业可视化图表：
   - 真实房价与预测房价对比散点图（直观观察拟合效果）
   - 房屋面积与房价线性回归趋势图
   - 全部特征相关性热力图（分析特征关联强弱）
   - 测试集预测误差分布柱状图；
7. 自动创建img文件夹保存高清图表，适配Windows中文显示，无乱码报错。

## 五、数据分析&建模结论
1. 特征相关性：房屋面积与房价正相关最强，距离地铁越远房价越低；
2. 线性回归模型拟合优度R²接近1，模型预测效果良好；
3. 面积每增加1㎡，房价显著上涨；地铁距离对房价存在负向影响。
